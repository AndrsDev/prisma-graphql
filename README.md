# prisma-graphql

Graphql API bulit with prisma, typescript and JWT.

## Get Started

1. `git clone git@github.com:AndrsDev/prisma-graphql.git`
2. `cd prisma-graphql`
3. `npm install`
4. `npm start`

### How to setup prisma
